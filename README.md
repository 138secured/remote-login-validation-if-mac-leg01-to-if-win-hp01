# IF-WIN-HP01 Endpoint Hardening Report

## Overview

This project documents a redacted endpoint hardening review for a Windows endpoint in the IronForge Manufacturing lab environment. The report focuses on baseline review, update validation, security posture improvement, and evidence-backed verification.

## Objectives

- Establish a baseline endpoint security posture
- Review installed applications and system status
- Validate Windows Update and security dashboard status
- Document hardening actions and verification evidence
- Produce a redacted portfolio-ready report
